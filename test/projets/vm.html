Installation de différentes machines virtuelles

Lorsque je suis arrivé à l'IUT de Lens, on nous a demandé de faire en sorte de pouvoir travailler sur linux sur nos machines personnelles. Nous avions le choix entre dual boot et vm. J'ai choisi les machines virtuels dans un premier temps parce que je ne voulais pas toucher au système de mon ordinateur. J'ai donc appris à utiliser des logiciels de virtualisation, à commencer par virtual box. Aujourd'hui je sais utiliser :

  • VirtualBox ;
  • Vmware workstation pro ;
  • QEMU KVM.
Et je sais virtualiser:
  • Windows xp ;
  • Windows 7 ;
  • Windows 8 ;
  • Windows 10 ;
  • Windows 11 ;
  • Ubuntu (la plupart des version) ;
  • Mint (la plupart des version) ;
  • Fedora (la plupart des version) ;
  • Arch ;
  • Macos Catalina ;
  • Macos Big sur ;
  • Macos Monterey.

Virtualiser ces os peut être une tache complexe, en particulier pour les vm macos (encore plus avec un processeur AMD). Mais cela me permet de pouvoir tester plusieurs travaux sur des os différents et d'utiliser des logiciels non compatibles avec linux.
