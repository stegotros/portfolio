Installation de différents systèmes d'exploitations

À cause de péripéties personnelles, j'ai du apprendre relativement tôt à installer un systèmes d'exploitation sur mon ordinateur. J'ai d'abord appris à installer Windows, donc à créer un clef bootable avec l'outil rufus. Puis ensuite j'ai appris à utiliser linux, j'ai donc aussi appris à faire des installations linux:

  • Ubuntu ;
  • Mint (cinnamon) ;
  • Fedora (pré-configuré à l'aide d'un script d'installation automatique);
  • Arch.

Aujourd'hui, pour un usage personnel, j'utilise uniquement linux mint.
